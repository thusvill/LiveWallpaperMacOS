FEATURES

Made to feel native.

Multiple Display Support

Separate configurations for each display. Your wallpapers adapt seamlessly across all your screens.

Optimized Performance

Under 20MB RAM usage with ~1% CPU load. Automatically pauses on lock screen to save battery.

Video Scaling Options

Fill, fit, or stretch your wallpapers. Support for .mp4 and .mov formats up to 4K resolution.

Volume Controls

Adjust audio levels for each wallpaper or mute completely. Full control over your experience.

Random Wallpapers

Set random wallpapers on startup or screen unlock. Keep your desktop fresh and surprising.

Native macOS Feel

Built with SwiftUI. Liquid glass effects that look and feel like they're part of macOS.

Installation

Homebrew

Quick install via package manager

brew tap thusvill/livewallpaper && brew install --cask livewallpaper

DMG Install

⬇ Download DMG

Download and drag to Applications

xattr -d com.apple.quarantine /Applications/LiveWallpaper.app

Build Source (Objective C++ branch)

Compile yourself

git clone https://github.com/thusvill/LiveWallpaperMacOS.git cd LiveWallpaperMacOS && mkdir build cmake .. && make -j$(sysctl -n hw.ncpu)
